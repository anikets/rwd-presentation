Intro

Responsive Web Design (RWD) is the black magic that helps Web developers build apps which are cross device and cross platform. Gone are the days of fixed 960 pixel width layouts. Long back. Today, viewers access websites from a number of devices. Responsive Web Design (RWD) is the answer to target all these users with a single app.

RWD apps support PC + Mac + Linux + Android + iOS + Windows Phone + Everything else with a Web browser.

RWD offers an enormous benefit to lean startups by enabling them to target a large number of users, as compared to developing native apps whose reach is limited to the selected platform.

What are media queries?

Media queries are CSS statements that facilitate applying CSS conditionally to elements. Device properties like viewport width, orientation, number of colours the device can display and type of device can be accessed through CSS and used to write conditional CSS.

A simple media query

@media (min-width: 700px) { ... }

A complex media query

@media (min-width: 700px), handheld and (orientation: landscape) { ... }

The picture element

<picture width="500"  height="500">
  <source  media="(min-width: 1000px)" src="large.jpg">
  <source  media="(min-width: 600px)" src="med.jpg">
  <source  src="small.jpg">
  <img  src="small.jpg" alt="">
  <p>Accessible  text</p>
</picture>
This is a work in progress and not currently supported in all browsers.
